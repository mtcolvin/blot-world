### Option 2: Manual JSON Creation

If you prefer to create the JSON manually or already have your poems in another format:

1. Create a `poems.json` file with this structure:
   ```json
   [
     {
       "id": "unique-poem-id",
       "title": "Your Poem Title",
       "date": "Month Year",
       "content": "Your poem content\nWith line breaks\nPreserved as \\n",
       "filename": "original-file.txt"
     }
   ]
   ```

2. Place `poems.json` next to `poetry-showcase-dynamic.html`

## URL Navigation

Once set up, you can link directly to specific poems:

- `poetry-showcase-dynamic.html` - Shows first poem
- `poetry-showcase-dynamic.html#page-1` - Shows first poem
- `poetry-showcase-dynamic.html#page-2` - Shows second poem
- `poetry-showcase-dynamic.html#page-3` - Shows third poem
- etc.

This is perfect for:
- Sharing specific poems via URL
- Creating a table of contents with direct links
- Integrating with your main portfolio site

## Customization

### Colors
Edit the CSS variables in the `<style>` section:
```css
:root {
    --parchment: #f4e8d8;      /* Main background */
    --ink: #2a2418;             /* Text color */
    --accent: #8b4513;          /* Accent color */
}
```

### Typography
Change the fonts by updating the Google Fonts link and CSS:
```css
body {
    font-family: 'Your Font', 'Courier New', monospace;
}
```

### Navigation Style
- Numbered buttons (current): Simple and clean
- Text labels: Change `btn.textContent = poem.title` in JavaScript
- Custom styling: Modify `.nav-btn` CSS class

## Troubleshooting

**"Unable to load poems" error:**
- Ensure `poems.json` is in the same directory as the HTML file
- Check browser console for specific errors
- Verify JSON is valid at [jsonlint.com](https://jsonlint.com)

**Poems not showing:**
- Check that poems array in JSON is not empty
- Verify JSON structure matches the example
- Look for JavaScript errors in browser console

**Hash navigation not working:**
- Ensure you're using `#page-1`, `#page-2` format (not `#1`, `#2`)
- Page numbers start at 1 (not 0)
- Check that the page number doesn't exceed your poem count
