Building BLOT.WORLD: A Modern Portfolio Experience

A deep dive into the design decisions, technical challenges, and creative process behind building my personal portfolio website from scratch.

Web Design HTML CSS JavaScript UX/UI

The Challenge

Every creative professional faces the same dilemma: how do you showcase your diverse skills and projects in a way that's both engaging and authentic? After exploring countless portfolio templates and platforms, I realized none captured what I wanted to communicate. I needed something unique, something that felt like me.

The goal was clear: create a digital space that serves as both a professional portfolio and a personal archive—a place where web design, branding projects, digital art, photography, and even poetry could coexist harmoniously.

Design Philosophy

Dark Mode by Default

I opted for a dark theme as the foundation. Not just because it's trendy, but because it allows colors, images, and interactive elements to truly pop. The deep black background (#0a0a0a) creates a gallery-like atmosphere where each project becomes a illuminated exhibit.

Typography as Expression

I chose Roboto Flex for its variable font capabilities. This single typeface can shift from condensed and sharp to wide and expressive, matching the tone of each section. The variable font axes give incredible flexibility without multiple font files weighing down the site.

Design Insight

Variable fonts aren't just about file size—they're about expression. A single typeface that can shift personality gives your design breathing room to adapt contextually.

Gradients & Depth

The purple-to-blue gradient became the visual signature throughout the site. It appears in buttons, headings, and interactive elements, creating visual continuity while adding a layer of sophistication. The gradient represents creativity flowing from concept to execution.

Technical Implementation

Vanilla JavaScript Philosophy

No frameworks. No build tools. Just clean, purposeful vanilla JavaScript. This decision was intentional:

  • Performance First: Zero framework overhead means faster load times and better performance
  • Full Control: Every animation, every interaction is hand-crafted
  • Learning Value: Understanding the fundamentals deeply before abstracting them away
  • Longevity: No dependency updates, no breaking changes

Project Filtering System

The filtering system was one of the more interesting challenges. I wanted users to filter by both area (Web Design, Branding, Photography, etc.) and technology (HTML, CSS, Affinity Designer, etc.) simultaneously, with real-time search functionality.

The solution uses a state management pattern that tracks active filters and applies them dynamically. Each project card contains data attributes that the JavaScript uses to determine visibility: 

<div class="project-card"
     data-area="Web Design"
     data-tech="HTML,CSS,JavaScript"
     data-name="Project Name">
</div>

Smooth Section Transitions

Rather than using traditional page navigation, I implemented a single-page application feel with smooth transitions between sections (Home, Projects, About). A brief loader animation creates anticipation and masks the content swap, making transitions feel intentional rather than jarring.

Unique Features

Inception Protection Modal

One of my favorite touches is the "Inception Protection" feature. When users try to click on the portfolio project card while viewing the portfolio, they get a playful Windows XP-style error message with a nostalgic error sound. It's a small Easter egg that adds personality and prevents the meta-confusion of viewing the portfolio within itself.

Animated Background Elements

The homepage features floating orbs, a grid pattern, and an interactive night sky canvas with twinkling stars. These elements create depth and movement without being distracting. They're subtle enough to enhance the experience rather than dominate it.

Responsive Design Considerations

Every element was designed to adapt gracefully across devices. The project grid adjusts from multi-column layouts on desktop to single-column on mobile, maintaining visual hierarchy and usability at every breakpoint.

Key Takeaways

  • Personality Matters: Small touches like the Inception Protection modal make your portfolio memorable
  • Performance is UX: Fast-loading sites keep visitors engaged and reflect your attention to detail
  • Constraint Breeds Creativity: Working without frameworks forced me to solve problems creatively
  • Your Portfolio is a Project: It should showcase not just what you've done, but how you think and solve problems

Challenges & Solutions

Challenge: Filter State Management

Managing multiple simultaneous filters (area + tech + search) while maintaining URL state proved tricky. The solution was creating a centralized AppState object that tracks all filter states and provides methods to reset, update, and apply filters consistently.

Challenge: Image Optimization

High-quality project previews can slow down the site. I implemented lazy loading for images, preloaded critical images (like the headshot and logo), and optimized all images before upload. The result: fast initial load with progressive enhancement.

Challenge: SEO for Single-Page App

Single-page applications can struggle with SEO since content loads dynamically. I ensured all content exists in the initial HTML, used semantic markup, and implemented proper meta tags. The JavaScript simply shows/hides existing content rather than creating it dynamically.

Lessons Learned

Building BLOT.WORLD reinforced several important principles:

  1. Start with Content: Design around your content, not the other way around. I cataloged all my projects first, then designed a system to showcase them.
  2. Progressive Enhancement: The site works with JavaScript disabled, but enhances with it enabled.
  3. Test Early, Test Often: Regular testing across devices and browsers caught issues before they became problems.
  4. Iterate Ruthlessly: The first version is never the final version. Be willing to tear down and rebuild.

What's Next

Like any good project, BLOT.WORLD is never truly "finished." Future enhancements include:

  • Adding this blog system for deeper dives into projects
  • Implementing analytics to understand how visitors engage with the site
  • Creating more interactive project showcases with embedded demos
  • Expanding the photography timeline with more interactive features

Your portfolio is more than a resume—it's a statement about who you are as a creator. BLOT.WORLD represents my belief that design should be intentional, technical execution should be clean, and personality should shine through in the details.

The best portfolio is one that reflects not just what you can do, but how you think. Make yours count.
